# CampusHub - 校园互助平台

## 1.项目简介

在校园生活中，学生之间存在大量互助需求：快递代取、学习辅导、物品交换、活动组队等。目前这些需求散落在微信群、QQ 群、贴吧等平台，信息碎片化、匹配效率低、缺乏信任机制。

CampusHub 旨在提供一个统一的校园互助服务平台，让学生能够便捷地发布需求、发现匹配、建立信任、完成协作。

### 核心功能

- **需求管理**
  - 发布互助需求
  - 需求分类
  - 需求列表浏览
  - 需求详情查看

- **智能匹配**
  - 基于分类、地理位置、时间的基础匹配
  - 基于用户历史行为的简单推荐

- **消息系统**
  - 站内消息
  - 简易即时通讯

- **后台管理**
  - 用户管理
  - 需求内容审核
  - 数据统计看板


## 2.技术栈

### 前端技术
- **Vue.js** - 渐进式 JavaScript 框架，用于构建用户界面
- **Axios** - 基于 Promise 的 HTTP 客户端，用于前后端数据交互

### 后端技术
- **Spring Boot** - Java 后端框架，提供 RESTful API 服务
- **MyBatis** - 持久层框架，实现数据访问层
- **MySQL** - 关系型数据库，存储业务数据
- **Maven** - 项目依赖管理和构建工具

### 开发工具
- **JDK 21** - Java 开发环境
- **Node.js** - 前端运行环境
- **Git** - 版本控制
- **IntelliJ IDEA / VS Code / WebStorm** - 集成开发环境
